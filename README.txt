Brief 5 - Les Jehdi des Maths

Ce brief a été réalisé de façon individuelle sur la période du 24/10/2022 au 26/10/2022.

Ce dernier a nottamment permis d'aborder les thématiques suivantes :
- le calcul par Python
- l'utilisation de la librairie Numpy
- les probabilités
- la visualisation
- les matrices

Les consignes sont directement sur les documents, permettant ainsi aux apprenants d'être guidés tout le long du brief et d'avancer en autonomie.
